한 줄 요약ひと言サマリー

"체성분을 객관적으로 비교하고 게임처럼 즐기게 만드는 컨셉" 자체는 일본 사용자에게 강하게 통했다. 다만 컨텐츠 정보량 과다, "클러스터" 단어의 부정적 이미지(감염 클러스터 연상), 목표 설정 후 구체적 행동 가이드 부재, UI 발견성 문제(각 항목 탭 인식 어려움)가 공통 페인포인트로 반복 등장한다.「体成分を客観的に比較し、ゲーム感覚で楽しめるコンセプト」自体は、日本のユーザーに強く刺さった。一方でコンテンツの情報量が多すぎる、「クラスター」という単語のネガティブな印象（感染クラスターを連想）、目標設定後の具体的なアクションガイドがない、UIの発見性問題（各項目をタップできることに気付けない）が共通の課題として繰り返し挙がった。

특히 Pathfinder는 "목표는 알겠는데 그래서 뭘 하면 되나?"라는 질문이 가장 많이 나왔다 — 운동/식단 추천, 외부 시설 연계, 단순/상세 모드 분리가 라이트 유저 확보의 핵심 레버다.特にPathfinderは「目標は分かったが、ではどう動けばよいのか？」という疑問が最も多く出た。運動・食事レコメンド、外部施設連携、シンプル／詳細モードの切り替えがライトユーザー獲得の鍵となる。

실행 우선순위 TOP 5 — 응답에서 가장 자주 반복된 요구実行優先度 TOP 5 — 回答に最も多く繰り返された要求

  1. 목표 설정 후의 행동 가이드 자동 연동 — Pathfinder에서 목표를 정한 뒤 식단/운동/추천 시설로 자연 흐름. (Q19/Q21/Q30에서 7명 이상 직접 언급)
    目標設定後のアクションガイド自動連動 — Pathfinderで目標を決めた後、食事・運動・推薦施設へ自然に繋がる導線。(Q19/Q21/Q30で7名以上が直接言及)
  2. "クラスター" 명칭 재검토 — 의학에서 부정적 의미(감염 클러스터). "분석", "체형 평가", "그룹" 등 대안 검토 요청 다수.
    「クラスター」名称の再検討 — 医療分野ではネガティブな意味（感染クラスター）。「分析」「体型評価」「グループ」など代替案の要望多数。
  3. 정보량 다이어트 + 단순/상세 모드 분리 — 라이트 유저에게는 압도적, 헤비 유저에게는 적절. 한 화면에 텍스트가 너무 많다는 지적 8건+.
    情報量ダイエット + シンプル／詳細モード分離 — ライトユーザーには圧倒的、ヘビーユーザーには適切。「1画面に文章が多すぎる」という指摘が8件以上。
  4. UI 발견성 개선 — Big Data에서 "각 항목 탭 시 상세 그래프"를 모르는 사용자 다수. ＞ 화살표/안내 텍스트 추가. 그래프 영역 활성화 기능도 동일.
    UIの発見性改善 — Big Dataで「各項目をタップすると詳細グラフが表示される」ことを知らないユーザー多数。＞ 矢印・案内テキスト追加。グラフ領域の活性化機能も同様。
  5. SNS 공유 기능 — MBTI처럼 "내 클러스터 공유" 요청 3건+. 자연 확산 레버로 가장 가성비 좋은 기능.
    SNSシェア機能 — MBTIのように「自分のクラスターをシェア」する要望3件以上。自然拡散レバーとして最もコスパが良い機能。

    データ使用量1.3GB過多 — テスト期間のみで計測。データ通信タイミングの手動設定オプション要望。
  • 2개 스마트폰에서 데이터 연동 시 잘못된 폰의 데이터가 동기화되는 버그 — A폰 → B폰 의도였으나 반대로 동작
    2台のスマホ連携で誤った端末のデータが同期するバグ — A端末→B端末を意図したが逆方向に動作
  • 홈 화면 수치(Fitscore, 체중, 골격근량, 체지방률) 최신값이 우측에 → 가독성 떨어짐
    ホーム画面の数値（Fitscore・体重・骨格筋量・体脂肪率）の最新値が右側 → 可読性低下
  • 홈 최상단 표시는 Fitscore보다 클러스터가 더 적절하다는 제안
    ホーム最上部表示はFitscoreよりクラスターが適切という提案
  • 해·달 마크 시간대 불일치 — 홈 화면은 12시간 구분, 측정 결과는 8시간 구분
    太陽・月マークの時間帯不一致 — ホーム画面は12時間区分、測定結果は8時間区分
  • 가정용 InBody와 자택 환경(Wi-Fi/Bluetooth) 연결 어려움 — "타사 제품을 샀다"는 응답까지 나옴
    家庭用InBodyと自宅環境（Wi-Fi/Bluetooth）の接続困難 — 「他社製品を買った」との回答まで
  • InBody와 앱 페어링 절차(개별 번호·Wi-Fi/Bluetooth)가 번잡
    InBodyとアプリのペアリング手順（個別番号・Wi-Fi/Bluetooth）が煩雑
  • DayLog의 입력 정확도 부족 — 다른 앱과 병용하게 됨
    DayLogの入力精度不足 — 他アプリと併用される
  • InBody Everyday 노출 강화 — 지속률을 높이는 매력적 기능인데 사용자 인지도가 낮음
    InBody Everyday の露出強化 — 継続率を高める魅力的機能だが認知度が低い

전략적 함의 — 일본 시장에서 "어떤 가치"가 가장 강하게 통했나戦略的含意 — 日本市場で「どの価値」が最も強く刺さったか

  1. "일본인 평균과의 비교"라는 가치 명제는 매우 강력하다. Big Data NPS +28.6, 그래프 비교 용이성 4.43/5 — 전 항목 중 최고. 일본 시장 메시징의 1순위 후크로 가져갈 만함.
    「日本人平均との比較」という価値提案は極めて強力。Big Data NPS +28.6、グラフ比較の分かりやすさ 4.43/5 — 全項目中の最高値。日本市場のメッセージングの第一の訴求点として活用できる。
  2. 클러스터의 "MBTI화" 시도는 성공했다. 단, "クラスター"라는 단어가 의학적으로 부정적 이미지(감염 클러스터)를 가진다는 일본 특유 이슈는 명칭 변경을 진지하게 검토할 만한 신호다.
    クラスターの「MBTI化」の試みは成功した。ただし「クラスター」という単語が医療的にネガティブな印象（感染クラスター）を持つという日本特有の課題は、名称変更を真剣に検討すべきシグナル。
  3. Pathfinder는 "비전은 좋지만 실행이 비어있다." 목표 설정 후 운동·식단·외부 시설 연계가 다음 우선순위. 이 갭을 메우면 NPS +2.4 → +20 이상 점프 가능성.
    Pathfinderは「ビジョンは良いが実行が空洞」。目標設定後の運動・食事・外部施設連携が次の優先課題。このギャップを埋めれば NPS +2.4 → +20以上 へジャンプの可能性。
  4. UI 발견성 문제는 디자인 미세조정으로 큰 효과를 볼 수 있는 영역. ＞ 화살표, 안내 텍스트, 그래프 영역 활성화 버튼 위치 같은 작은 변경으로 "기능을 알지도 못한 채 떠나는 사용자"를 줄일 수 있다.
    UI発見性の問題は、デザインの微調整で大きな効果が見込める領域。＞矢印、案内テキスト、グラフ領域活性化ボタンの位置といった小さな変更で「機能を知らないまま離脱するユーザー」を減らせる。
  5. 업무용 InBody 데이터 연동은 사내 사용자 신뢰 확보의 게이트키퍼. 실제 출시 전 우선순위에 두지 않으면 일본 법인 내부 옹호자(advocate)부터 잃는다.
